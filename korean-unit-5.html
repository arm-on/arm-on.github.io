اعداد ترکیبی |--|--|--| | کلمه/عبارت | ترجمه | تلفظ | |--|--|--| | 열 하나 | یازده / 10+1 | | | 열 둘 | دوازده | | | 열 셋 | سیزده | | | 스물 둘 | بیست و دو | | | 쉰 셋 | پنجاه و سه | | | 아흔 둘 | نود و دو | | | 서른 여덟 | سی و هشت | | - اول دهگان رو میگی بعد یکان. خیلی ساده - دقت کن که صد و هزار در واقع چینی هستن. پادشاه سجونگ فقط اعداد ۱ تا ۹۹ رو به کره‌ای ترجمه کرده. اعداد بزرگتر رو از چینی استفاده می‌کنن. نکته: اگر «د» در جایگاه پدچیم داشته باشیم و بعدش «س» بیاد، خیلی کره‌ای تره که یه «س» مشدد تلفظ کنیم. مثلا 몇 살이에요 رو بخونیم «میُسَل‌ای‌اِیُ». # گرامر ## گفتن سن نکته: اعداد ۱ و ۲ و ۳ و ۴ هرگاه در یکان قرار بگیرند و بعدش واحد شمارش بیاد، فرم نوشتاری‌شون تغییر میکنه. حرف آخرشون حذف میشه. مثال: - 하나 + 명 = 한 명 (یک نفر) - 하나 + 살 = 한 살 (یک سال) - 둘 + 명 = 두 명 (دو نفر) - 셋 + 명 = 세 명 (سه نفر) - 넷 + 명 = 네 명 (چهار نفر) برای اعداد بیشتر از ۴، فرم نوشتاری‌شون تغییر نمی‌کنه. مثال: - 다섯 + 명 = 다섯 명 (پنج نفر) - 여섯 + 명 = 여섯 명 (شش نفر) - 일곱 + 명 = 일곱 명 (هفت نفر)
